Epic Crypto Wars

Ethereum Dapp

Epic Crypto Wars is a real-time multiplayer game made in Unity using blockchain technologies. The game consists of two parts. The first part is multiplayer in Unity (game mechanics) and Photon (networking). The second part is a decentralized application (dapp) on Ethereum network. In a game, players can collect, trade, and buy heroes (cards). With heroes build an army and fight with other players. The battle is PvP, one on one and before a battle, both players chose an army of heroes with whom they will fight. All the player's data is stored on Ethereum network, currently the network is running locally and used on test network. Soon we will put our game on a real network and give everyone a chance to play.

  • Technologies: C#, Unity, Photon, Solidity, Blockchain, Ethereum, Ganache, Truffle

Mystery Island

Puzzle Game

Mystery Island is a third-person adventure puzzle game. In a game, you play as the main character who is trapped in his dream. The dream consists of many mysterious islands in which player need to find a way out. Every island has a magic portal that can be open after solving some puzzles. Portal serves as a passage to the next island.

  • Technologies: C#, Unity, Blender, Python

Dream Back

RPG Game

DreamBack is an RPG simulation game built in Unity. Game is a simulation in which player can explore medieval history and architecture and also fight with other characters in a game.

  • Technologies: C#, Unity, Blender

STL Nails

Nail Salon Software

STL Nails is a web application for booking online appointments in a beauty salon. It allows you to add and display gel nail works so that users can have an insight into the work of the salon. Users can book an appointment for the service offered by the salon, they can see the prices of the salon, and if there is any ambiguity, they can contact the salon for an inquiry.

  • Technologies: PHP, JavaScript, MySQL, XAMPP, HTML, CSS

Financis

Budget App

Financis is a web application for tracking family / group costs. Provides insight into the daily, weekly, or monthly spending of an individual or group. The user has the option of displaying statistics by cost category, each member of the group as well as comparing monthly cost differences.

  • Technologies: Python, Flask, Vue.js, SQLite, Pony ORM

University Accessories

Web Shop

University Accessories is a web application for ordering University articles. Provides the admin with the addition of new products, prices, descriptions, etc. The user can add the product to the cart and make payment for it and receive your order confirmation by mail.

  • Technologies: PHP, JavaScript, HTML, CSS, MySQL
